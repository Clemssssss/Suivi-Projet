Paramétrage du fichier

Configure ici l’URL Excel SharePoint à synchroniser vers la base du dashboard. Le refresh réimporte ensuite chaque projet dans la DB.
Chargement…
Ce mode fonctionne immédiatement si tu fournis une vraie URL Excel téléchargeable par le serveur Netlify. Si ton lien SharePoint ouvre seulement Office/SharePoint, utilise le mode Lien de partage SharePoint + Graph avec un bearer token Microsoft Graph.
Comment remplir les champs
1. URL du fichier Excel SharePoint : colle soit une URL de téléchargement direct, soit ton lien de partage SharePoint normal.
2. Nom de source : libellé libre affiché dans le suivi.
3. Dataset key : laisse saip-main si tu veux mettre à jour le dashboard principal.
4. Mode d'authentification :
- Aucun header d'auth si le lien est publiquement téléchargeable.
- Bearer token si tu as déjà une vraie URL de fichier protégée.
- Lien de partage SharePoint + Graph si tu colles un lien SharePoint standard comme /:x:/r/....
5. Activation : active la source pour autoriser le refresh automatique.

Ordre conseillé : Sauvegarder → Tester la source → Refresh SharePoint.
